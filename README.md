# Heaven-s-Feel-Simulator

성배전쟁을 시뮬레이션합니다.

## How to run

### Prerequisite
- [Node.js](https://nodejs.org/) 18 or higher

### Run

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

Connect to `http://localhost:5173`

### Build Production

```bash
npm run build
```

`dist/` 폴더에 정적 파일이 생성됩니다.
